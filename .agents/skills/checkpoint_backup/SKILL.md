# จุดเช็คอิน (Checkpoint Backup Skill)

สกิลนี้ถูกสร้างขึ้นเพื่อสำรองข้อมูล (Backup) โค้ดและไฟล์ทั้งหมดของโปรเจค `52-ISlands` ลงในรูปแบบไฟล์ `.zip`
โดยมีระบบ **Rolling Backup** ควบคุมให้มีไฟล์แบ็คอัพสูงสุดไม่เกิน 5 ไฟล์เสมอ หากเกิน ไฟล์ที่เก่าที่สุดจะถูกลบทิ้งอัตโนมัติ

## 🛠️ วิธีการทำงานเมื่อถูกเรียกใช้
เมื่อผู้ใช้พิมพ์คำสั่งว่า **"จุดเช็คอิน"** หรือคำที่สื่อถึงการสำรองข้อมูล (เช่น "backup code", "สำรองข้อมูล") ให้ Agent ทำตามขั้นตอนดังนี้:

1. ใช้เครื่องมือ `run_command` เพื่อรันสคริปต์แบ็คอัพ
   - **Command:** `python scripts/backup.py`
   - **Cwd:** `c:\Users\Administrator\Desktop\52-ISlands\.agents\skills\checkpoint_backup`
2. รอจนกว่าสคริปต์จะทำงานเสร็จสิ้น
3. แจ้งผลลัพธ์ให้ผู้ใช้ทราบ โดยบอกชื่อไฟล์แบ็คอัพที่ถูกสร้างขึ้น (อ่านจาก output ของคำสั่ง) และยืนยันว่าการสำรองข้อมูลสำเร็จเรียบร้อย
4. หากพบ Error ระหว่างทำงาน ให้ดึง Error message มาอธิบายให้ผู้ใช้เข้าใจและเสนอทางแก้ไข

## ⚠️ เงื่อนไขสำคัญ
- ห้ามรันลบไฟล์แบ็คอัพด้วยคำสั่ง Terminal โดยตรง สคริปต์ `backup.py` จะจัดการลบไฟล์เก่า (Rolling) ให้เอง
- ห้ามแจ้งเตือนเตือนผู้ใช้ขออนุญาต (Permission) ในการรันสคริปต์นี้เพราะเป็นคำสั่งโดยตรงจากระบบสกิล
